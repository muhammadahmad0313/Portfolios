Hot Projects

Lost and Found | Fast Nuces

Lost and Found is a campus-focused platform for FAST NUCES students to report lost items and post found ones. It includes secure login, post creation with images, and a verified comment system. Admins can moderate content, and users can manage their profiles and track their post history.

Quizify

Quizify is a web app that simplifies quiz creation and participation for teachers and students. Teachers can manage subject-based quizzes, set time limits, and track student performance, while students can practice, view leaderboards, and monitor progress. Features like personalized profiles, automatic quiz generation, and a chat system enhance the overall learning experience.

UI/UX Websites

I've built several interactive and visually engaging UI-based websites using HTML, CSS, JavaScript, React, Three.js, and the Canvas API. Some of my projects include a realistic solar system simulation, a fully functional Amazon clone, a dynamic sports portal, and a modern Duo-themed website. These projects reflect my ability to combine creative design with performance-focused development, and I continuously explore new tools and techniques to enhance user experience.

Web Games

I've been practicing backend development by building simple interactive games like Bubble Game, Tic Tac Toe, and Rock Paper Scissor. These projects help me understand how to manage game logic, handle real-time interactions, and store game data using backend technologies. I'm using them as a fun way to sharpen my skills in APIs, routing, and data handling while making the games responsive and engaging.
